# [シーラブ] 



人魚を操作して宝石を集めるネット対応マルチアクションゲームです。
2年次のゲームゼミにて、主にJoy-Con対応を担当しました。

## 🎮 ゲーム概要
- **ジャンル**: 3Dマルチアクション
- **プラットフォーム**: PC (Windows)
- **制作期間**: 約5ヶ月
- **言語**: C#
- **使用ツール**: Unity [バージョン： 2022.3.42f1], VisualStudio2022, git, GitHub, Sourcetree

## 🌟 実装のポイント
- **ジャイロ操作**: 取得したジャイロ値を補正することで、急な回転を防ぐようにしました。
- **再接続**: ゲーム中にJoy-Conの接続が切れた場合にプログラム上で再接続を行うようにしました。

## 💡 こだわった点 
- **エフェクトとアニメーション**: ターゲットユーザーが小学生のため、常に画面内に動きがあるものを置くようにしました。
